Detailed Description

UVBus v2 - Minimal Transport Abstraction Layer.

Design Principles:

  • Minimal: Minimized API, each function does one thing
  • Zero threads: All operations execute asynchronously in event loop
  • Zero locks: Single-threaded model (except INPROC)
  • Zero global variables: All state passed through context
  • Loop injection: Does not manage loop lifecycle, fully controlled by user

Loop Injection Design:

  • UVBus accepts user-provided uv_loop_t* pointer
  • UVBus does not create, start, or stop loop
  • UVBus does not free loop
  • User has full control over loop lifecycle
  • Supports multiple instances, multiple loops, loop reuse

Responsibilities:

  • Pure transport layer (TCP/UDP/IPC/INPROC)
  • Byte send/receive
  • Connection management
  • Does not handle business logic
