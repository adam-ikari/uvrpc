\doxysubsection{Detailed Description}
UVBus v2 -\/ Minimal Transport Abstraction Layer. 

Design Principles\+:
\begin{DoxyItemize}
\item Minimal\+: Minimized API, each function does one thing
\item Zero threads\+: All operations execute asynchronously in event loop
\item Zero locks\+: Single-\/threaded model (except INPROC)
\item Zero global variables\+: All state passed through context
\item Loop injection\+: Does not manage loop lifecycle, fully controlled by user
\end{DoxyItemize}

Loop Injection Design\+:
\begin{DoxyItemize}
\item UVBus accepts user-\/provided uv\+\_\+loop\+\_\+t$\ast$ pointer
\item UVBus does not create, start, or stop loop
\item UVBus does not free loop
\item User has full control over loop lifecycle
\item Supports multiple instances, multiple loops, loop reuse
\end{DoxyItemize}

Responsibilities\+:
\begin{DoxyItemize}
\item Pure transport layer (TCP/\+UDP/\+IPC/\+INPROC)
\item Byte send/receive
\item Connection management
\item Does not handle business logic
\end{DoxyItemize}
