\doxysubsection{Detailed Description}
Functions for creating and configuring UVRPC instances. 

IMPORTANT\+: Request/\+Response Data Lifetime

Server Request (uvrpc\+\_\+request\+\_\+t)\+:
\begin{DoxyItemize}
\item req-\/\texorpdfstring{$>$}{>}method and req-\/\texorpdfstring{$>$}{>}params point to frame data and are ONLY VALID during the handler callback
\item Do NOT store these pointers for later use -\/ they will be freed after the callback returns
\item If you need to use the data later, make a copy using \mbox{\hyperlink{group__AllocatorAPI_ga73a86be88aff6f12142368fced3605f2}{uvrpc\+\_\+strdup()}} for method or \mbox{\hyperlink{group__AllocatorAPI_ga40028cd95db0b1659f90e48e1d732905}{uvrpc\+\_\+alloc()}} + memcpy() for params
\item req-\/\texorpdfstring{$>$}{>}client\+\_\+stream can be used to send responses during the callback
\end{DoxyItemize}

Client Response (uvrpc\+\_\+response\+\_\+t)\+:
\begin{DoxyItemize}
\item resp-\/\texorpdfstring{$>$}{>}result points to copied data that is valid until the callback returns
\item The framework automatically frees resp-\/\texorpdfstring{$>$}{>}result after the callback
\item Do NOT store resp-\/\texorpdfstring{$>$}{>}result for later use -\/ make a copy if needed
\item Use \mbox{\hyperlink{group__RequestResponseAPI_gafb7e16788fb623e8ae12a671aaa2a99d}{uvrpc\+\_\+response\+\_\+free()}} only if you allocated the response structure yourself (rare) 
\end{DoxyItemize}
