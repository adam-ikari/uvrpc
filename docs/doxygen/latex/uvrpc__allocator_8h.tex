\doxysubsection{Detailed Description}
UVRPC Memory Allocator. 

Provides a flexible memory allocation interface supporting three modes\+:
\begin{DoxyItemize}
\item system\+: Uses standard malloc/free (best compatibility)
\item mimalloc\+: Uses mimalloc high-\/performance allocator (default)
\item custom\+: Uses user-\/provided custom allocator
\end{DoxyItemize}
