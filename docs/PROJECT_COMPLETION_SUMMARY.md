## Architecture Highlights

### Design Philosophy
- Zero threads, zero locks, zero global variables
- Event-driven architecture using libuv
- Loop injection for multi-instance support
- Multiple transport protocols (TCP, UDP, IPC, INPROC)
- Zero-copy where possible

### Performance Characteristics
- INPROC: 125,000+ ops/s, 0.03ms latency
- IPC: 91,895 ops/s, 0.10ms latency
- UDP: 91,685 ops/s, 0.15ms latency
- TCP: 86,930 ops/s, 0.18ms latency
